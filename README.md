# Netflix--Data-analysis--powerbi-dashboard
🎬 Netflix Data Analysis Dashboard

📊 Overview

This project analyzes Netflix dataset using Microsoft Power BI to uncover trends in content, genres, and ratings.

🚀 Tools Used

- Microsoft Power BI
- Power Query

📈 Insights

- Movies vs TV Shows distribution
- Genre analysis
- Year-wise content trends
- Ratings breakdown
